Installation

Dans une VM Debian Wheezy 64bits(*) sous ProxMox :

La http://www.packetfence.org/support/faqs/article/how-to-install-packetfence-on-debian.html | doc] 

 echo "deb http://inverse.ca/downloads/PacketFence/debian wheezy wheezy" >> /etc/apt/sources.list

Ajouter non-free dans le dépôt Wheezy : deb http://ftp.debian.org/debian wheezy main contrib non-free 

 apt-key adv --keyserver keys.gnupg.net --recv-key 0x810273C4
 apt-get update
 apt-get dist-upgrade
 apt-get install ca-certificates

Indispensable au préalable sinon vous avez une belle erreur car le paquet freeradius pose souci sans ca-certificates installé au préalable (32 comme 64b) 

 apt-get install packetfence

Puis fire your web browser to https://IP:1443/configurator
