Le squeezebox touch a gagné. Alors, plutôt qu'un géant américain de la vente à distance, j'ai choisi un petit revendeur français, lyonnais : Eric Boisseau; à peine plus cher que l'autre, mais français, compétitivité, protectionnisme,... tout ça tout ça .


Maintenant après déballage, il faut configurer le biniou et ...en fait on ne peut pas passer par un simple partage réseau SMB ou NFS, il faut installer un bout de software sur le NAS.

Installation Logitech Media Server

FreeBSD

Bon, y'a un port FreeBSD : 

cd /usr/ports/audio/squeezeboxserver
make config install clean

(Long, long long très long) 

echo 'squeezeboxserver_enable="YES"' >> /etc/rc.conf 
/usr/local/etc/rc.d/squeezeboxserver start


Debian

Doc 

wget http://downloads.slimdevices.com/nightly/7.9/sc/61d60ba/logitechmediaserver_7.9.1~1497530649_amd64.deb
dkpg -i logitechmediaserver_7.9.1~1497530649_amd64.deb

Connexion distante SSH

  • Activer SSH sur le boitier (mdp 1234 par défaut)
ssh root@IP
mkdir -p /root/.ssh
exit
scp /home/phil/.ssh/id_rsa.pub root@IP:/tmp
ssh root@IP
mv /tmp/id_rsa.pub /root/.ssh/authorized_keys
exit

Vérifier que l'on peut se connecter sans password 

 ssh root@IP
passwd root
