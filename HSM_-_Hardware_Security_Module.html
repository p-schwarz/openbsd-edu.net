Principes

Une mémoire Write-Only (peut être écrite, mais pas lue) stocke les clés symétriques. Lors d'une demande de validation d'un élément d'authentification (clé, OTP, hachage,..), on soumet la requête au HSM qui répond OK/NOK. Les clés maîtresses permettant la validation ne sortant jamais du boitier, il est devrait être impossible de les récupérer pour réaliser une attaque de l'homme du milieu. Bref, votre serveur qui s'est fait compromettre ne permettra pas, même à root/Unix ou System/Windows d'accéder aux clés.

Afin de réduire la surface d'attaque, et vu la faible bande passante nécessaire, le HSM est vu comme un périphérique de type série (port COMx ou bien /dev/ttyACMx)accessible via minicom (sous Unix) ou Terminal (Windows).

Corollaire : Pas besoin de driver pour causer au module!

Le mode de configuration se fait en insérant la clé USB tout en appuyant sur le micro-interrupteur dans le trou d'épingle.

Fonctionnalités

  • MAC : Message Authentication Code
  • Fonctions de hachage : HMAC-SHA1
  • {Dé}chiffrement AES
  • Vrai (non pseudo) générateur d'aléa TRNG : True Random Number Generation.

Vocabulaire

  • Master key : Mot de passe facultatif permettant le déverrouillage du module. Utilisé pour sécuriser le boot du HSM (qui peut se faire voler et démarrer ailleurs).

Clés et fonctions principales

Clés

  • Une clé avec ID=1 sera utilisée pour réaliser du hachage de mots de passe.
  • Une clé avec ID=2 sera utilisée pour chiffrer/déchiffrer d'autres éléments

Fonctions

A minima, il faut utiliser les fonctions suivantes :

  • YSM_NULL
  • YSM_SYSTEM_INFO_QUERY
  • YSM_ECHO
  • YSM_AES_ECB_BLOCK_ENCRYPT
  • YSM_AES_ECB_BLOCK_DECRYPT
  • YSM_HMAC_SHA1_GENERATE
  • YSM_RANDOM_GENERATE
  • YSM_RANDOM_RESEED

Warning, Warning

Si vous basez votre sécurité sur ce HSM,...achetez-en deux ! En cas de perte du premier, et sous réserve que vous ayez copié les clés sur le HSM-backup, vous pourrez continuer à fonctionner....

On verra ce qui peut être réalisé sur ce backup vers le yubiKSM (HSM Soft)

Installation

apt-get install python-pyhsm minicom python-setuptools
sudo wget https://github.com/Yubico/python-pyhsm/archive/master.zip
sudo unzip master.zip 
cd python-pyhsm-master/


  • Insérez le HSM dans le port USB tout en maintenant avec un trombone le micro interrupteur enfoncé...Faut du doigté ;-)
sudo minicom (port /dev/ttyACM0)
