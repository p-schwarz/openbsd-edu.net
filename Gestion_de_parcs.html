Dépôt local de paquets

La doc évoluée

  • Sur le serveur
apt-get install apt-cacher-ng
echo "AdminAuth: utilisateur:motdepasseblinde" >>  /etc/apt-cacher-ng/security.conf
service apt-cacher-ng restart

Evidemment, il faut que le serveur ait une config IP/DNS correcte!

  • Sur les clients
echo 'Acquire::http { Proxy "http://apt:3142"; };' | sudo tee /etc/apt/apt.conf.d/01apt-cacher-ng-proxy

où apt est le nom local du serveur

Une boulette pour les clients Ubuntu : 

echo "http://fr.archive.ubuntu.com/ubuntu/ " >> /etc/apt-cacher-ng/backends_ubuntu
service apt-cacher-ng restart
  • Surveillance du serveur : http://apt:3142/acng-report.html

On voit que lors de la première maj, le cache est inefficace, ne contenant aucune donnée; ce qui n'est plus le cas après.

Config

/etc/apt-cacher-ng/acng.conf r-gle la conf du soft, en particulier :

  • CacheDir: /var/cache/apt-cacher-ng
