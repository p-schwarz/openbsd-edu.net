Monter un disque FreeBSD sous Linux

sudo mount -r -o ufstype=ufs2   -t ufs /dev/sdd1 /mnt/ufs/

un petit 

apt-get install ufsutils au préalable..

Monter un disque linux sous FreeBSD

pkg install e2fsprogs

puis 

mount -t ext2fs /dev/da4s3 /mnt

Si le montage ne se fait pas et qu'un dmesg donne 

WARNING: mount of da4s1 denied due to unsupported optional features

C'est juste que seules les partitions ext2 & ext3 sont utilisables,pas ext4...

ZFS

Formater un disque

  • Créer le schéma de partition
gpart create -s gpt ada4
  • Voir le schéma de partition 
gpart show ada3
=>       63  156301425  ada3  MBR  (75G)
        63       1985        - free -  (993K)
      2048  156297216     1  ntfs  (75G)
 156299264       2224        - free -  (1.1M)
  • Supprimer une partition
gpart delete -i 1 ada3
ada3s1 deleted

La partition NTFS précédente est ..vanished..

  • Détruire une table de paritition
gpart destroy ada3
ada3 destroyed

Créer une partition

gpart add -t freebsd-zfs -l zil -s 40G ada3
ada3p1 added

Debug

zpool add datas log ada3s1
 cannot open 'ada3s1': no such GEOM provider
 must be a full path or shorthand device name

zpool add datas log /dev/ada3p1

Identifier un disque

A part le 

dmesg |grep ada

On a 

camcontrol identify ada3
