Marionnet

De OpenWikiBSD
Aller à : navigation, rechercher

C'est un logiciel de simulation réseau didactique. C'est à dire à destination des élèves, étudiants et qui n'est pas Cisco-orienté comme l'est Gns3. Développé par une univsertisté française, seule l'installation est pénible :

La doc d'install: [[1]]


Installation sur Ubuntu 12.04

 sudo apt-get update && sudo apt-get dist-upgrade
  • En I386 : 
sudo apt-get install -y libdevplug2 bzr  ocaml-native-compilers ocaml-compiler-libs camlp4\*  liblablgtk2-gnome-ocaml-dev graphviz uml-utilities  bridge-utils gcc xserver-xephyr make gettext xterm  g++  autoconf automake  libtool
  • En Amd64 : 
sudo apt-get install -y ia32-libs bzr  ocaml-native-compilers ocaml-compiler-libs camlp4\*  liblablgtk2-gnome-ocaml-dev graphviz uml-utilities  bridge-utils gcc xserver-xephyr make gettext xterm  g++  autoconf automake  libtool

 cd /usr/src/
 sudo tar  -xjf /home/ps/marionnet-version-iut.tar.bz2 
 sudo apt-get remove --purge vde2
 cd marionnet-version-iut/vde
 sudo autoreconf -fi
 ./configure &&  make && sudo make install 
 cd ../ocamlbricks 
 make && sudo make install
 cd ../marionnet 
 make && sudo make install
 cd /usr/local/share
 sudo tar -xjf /home/ps/image-marionnet-2012-11-11.tar.bz2 
 sudo chmod a+rw /dev/net/tun 
 sudo /usr/local/sbin/marionnet-daemon.native &> /dev/null & 
 sudo /etc/rc.local

Puis enfin pour lancer le logiciel : 

/usr/bin/env UBUNTU_MENUPROXY=0 /usr/local/bin/marionnet.native

Configuration

  • Il ne faut pas toucher à la config IP des machines par leur console. Il faut utiliser l'onglet interfaces de celles-ci. Sinon, elles ne fonctionneront pas.
  • Les identifiants es machines sont root/root
  • Vous ne pouvez pas enregistrer votre réseau si l'un quelconque des objets est actif (loupiotte verte)
