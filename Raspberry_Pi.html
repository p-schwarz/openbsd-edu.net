Réseau

Le pi (version Raspbian ) utilise par défaut dhcpcd.conf. Pour le passer en IP fixe : Editer /etc/dhcpcd.conf 

# Custom static IP address for eth0.
interface eth0
static ip_address=192.168.0.200/24
static routers=192.168.0.1
static domain_name_servers=192.168.0.1

Debug

/lib/modules/3.10.25+/build: No such file or directory.  Stop.

Il faut ajouter des modules [3] 

    sudo apt-get install linux-headers

choisir le dernier module 

   sudo apt-get install linux-headers-3.10-3-rpi
   sudo ln -s /lib/modules/3.10-3-rpi/build /lib/modules/3.10.25+/build

Camera

La raspicam est minuscule, de bonne qualité et pas chère

Debug

Avec motion, vous n'obtenez pas de flux ,mais "unable to open video device"

Lancer motion à la main 

Failed to open video device /dev/video0: No such file or directory

La solution

Ajouter à votre sources.list 

deb http://www.linux-projects.org/listing/uv4l_repo/raspbian/ wheezy main

wget http://www.linux-projects.org/listing/uv4l_repo/lrkey.asc && sudo apt-key add ./lrkey.asc
apt-get update
apt-get dist-upgrade
sudo apt-get install uv4l uv4l-raspicam
uv4l --driver raspicam --auto-video_nr
sudo apt-get install motion
sudo LD_PRELOAD=/usr/lib/uv4l/uv4lext/armv6l/libuv4lext.so motion -c /etc/motion/motion.conf
